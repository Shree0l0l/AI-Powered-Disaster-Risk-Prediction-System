# AI-Powered-Disaster-Risk-Prediction-System
An end-to-end machine learning project that predicts disaster-prone regions in India, estimates impact (population, fatalities, economic loss), and generates risk scores using historical and geospatial data.
An AI-based risk mapping system that identifies disaster-prone regions across India using historical disaster data and geospatial analysis. The project focuses on predicting disaster hotspots, estimating impact, and analyzing risk patterns to support better disaster management.

**Features**

**1.Disaster Hotspot Prediction:**
Identify high-risk regions across India, 
Detect hotspots for:
Floods,
Cyclones,
Earthquakes,
**2.Disaster Impact Prediction:**
Predict disaster severity using machine learning models
Estimate:
  Affected population,
  Human loss (deaths),
  Economic damage,
  Analyze historical disaster trends,
  Calculate risk scores
**3.Disaster Pattern & Risk Score Analysis**
Identify patterns in disaster occurrence over time,
Rank regions into:
  Low Risk,
  Medium Risk,
  High Risk
